# Chimera
Your personal file headed monster.

## Origin Story
The purpose of *Chimera* is to combine files together into a singular file. When creating this program, I found a number of programs that would bundle code into an executable. My use case is to bundle programming files into a singular source code file intelligently. I have a library of useful snippets that I want to pull into a singular file. This works great for languages like Python, Rust, or Elixir, all of which adhere to a more scripty nature.
